Python Data Types



What are Data Types ?

In programming, data types define the type of data that a variable can hold. They determine the nature of the data and the operations that can be performed on it. Data types are crucial for ensuring that the computer's memory is used efficiently and that the operations performed on the data make sense.

Data Types in Python :

Python is a dynamically-typed language, which means that the interpreter automatically determines the data type of a variable at runtime.

Numeric Types :

int :

int : Integer type represents whole numbers without any decimal points. 

age = 25
x = 100
float :

float : Float type represents real numbers with a decimal point. 

height = 5.8
age = 25.01
complex :

complex : Complex numbers with a real and imaginary part. 

complex_num = 3 + 5j

Text Types :

int :

int : Integer type represents whole numbers without any decimal points. 

name = "Wel come to C2Zone"
name = "C2Zone"

Sequence Types :

list :

list : Lists are mutable sequences, allowing the storage of different data types. 

my_list1 = [  1,  2  ,"C"  ,"2"  ,"Zone"]
my_list1 = [ "Wel-Come","to", "C2Zone"]
tuple :

tuple : Tuples are immutable sequences. 

coordinates = (3, 4)
range :

range : Represents a sequence of numbers and is often used in loops. 

numbers = range(1, 5)

Set Types :

set :

set : Sets are unordered collections of unique elements. 

my_set = {1, 2, 3}
frozenset :

frozenset : An immutable version of a set. 

frozen_set = frozenset([4, 5, 6])

Mapping Types :

dict :

dict : Dictionaries are collections of key-value pairs. 

person = {"name": "Alice", "age": 30}

Boolean Types :

bool :

bool : Represents boolean values, `True` or `False`. 

is_adult = True
isMature = False

None Types :

None :

none : Represents the absence of a value or a null value. 

result = None
