Python Operators



What are Operators ?

Operators in Python are symbols that perform operations on operands. They are fundamental to programming as they allow you to manipulate and work with data.

Arithmetic Operators :

Arithmetic Operators : Perform basic mathematical operations.

Example : 

addition = 10 + 5
subtraction = 10 - 5
multiplication = 10 * 5
division = 10 / 5
modulus = 10 % 3
exponentiation = 10 ** 2

Comparison Operators :

Comparison Operators : Compare values and return Boolean results.

Example : 

equal = (5 == 5)
not_equal = (10 != 5)
greater_than = (10 > 5)
less_than = (10 < 5)
greater_than_equal = (10 >= 10)
less_than_equal = (10 <= 5)

Assignment Operators :

Assignment Operators : Assign values to variables.

Example : 

x = 10

                        z *= 2  # Equivalent to z = z * 2

Logical Operators :

Logical Operators : Combine Boolean values and return Boolean results.

Example : 

logical_and = (True and False)
logical_or = (True or False)
logical_not = not True

Identity Operators :

Identity Operators : Compare the memory location of two objects.

Example : 

a = [1, 2, 3]
b = [1, 2, 3]
is_equal = (a is b)  # False
is_not_equal = (a is not b)  # True

Membership Operators :

Membership Operators : Test whether a value is a member of a sequence.

Example : 

my_list = [1, 2, 3]
in_list = (2 in my_list)  # True
not_in_list = (4 not in my_list)  # True

Bitwise Operators :

Bitwise Operators : Perform bitwise operations on integers.

Example : 

bitwise_and = 10 & 5
bitwise_or = 10 | 5
bitwise_xor = 10 ^ 5
bitwise_not = ~10
left_shift = 10 << 2
right_shift = 10 >> 2

These operators are fundamental for performing various operations and comparisons in Python. Understanding how to use them effectively is crucial for writing expressive and functional code.
