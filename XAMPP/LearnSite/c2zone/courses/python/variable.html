Python Variables



Basics of Python Variables :

In Python, a variable is a symbolic name that refers to a value. Variables are used to store and manage data in your program. Here are some key points about Python variables:

Variable Assignment :

To create a variable, you use the assignment operator (=). The variable name is on the left, and the value to be stored in the variable is on the right. 

age = 25
name = "John"

Variable Naming Rules :

Variable names can contain letters, numbers, and underscores. They cannot start with a number. Python is case-sensitive. 

my_variable = 42
age1 = 30

Data Types:

Python is dynamically typed, meaning you don't need to declare the type of a variable explicitly. The interpreter infers the type based on the assigned value.

my_integer = 42
my_float = 3.14
my_string = "Hello, Python!"

Advanced Concepts :

Dynamic Typing :

Python is dynamically typed, allowing you to reassign variables to different types. 

age = 25
age = "Twenty-Five"

Multiple Assignments :

You can assign multiple variables in a single line.

x, y, z = 10, 20, 30

Variable Swapping :

Python allows for a concise way to swap the values of variables.

a, b = 5, 10
a, b = b, a

Constants :

While Python doesn't have constants in the traditional sense, it is a convention to use uppercase names for variables that should be treated as constants

PI = 3.14159

None :

In Python, None is a special constant representing the absence of a value or a null value.

result = None

Global vs. Local Variables :

Variables declared outside a function or a block are global, while those declared inside are local. Global variables can be accessed from any part of the code, but local variables are limited to their scope.

global_var = 10

def my_function():
    local_var = 5
    print(global_var)

Variable Scope :

Python follows the LEGB (Local, Enclosing, Global, Built-in) rule when searching for the value of a variable.

x = 10

def my_function():
    x = 5
    print(x)

my_function()
    print(x)

Mutable vs. Immutable Types :

Some data types in Python, like lists, dictionaries, and sets, are mutable, meaning their values can be changed. Others, like strings and tuples, are immutable.

mutable_list = [1, 2, 3]
immutable_tuple = (1, 2, 3)

mutable_list.append(4)
# Valid operation # immutable_tuple.append(4)  # This will raise an error
